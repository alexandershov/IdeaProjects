## Zig

### What is it?

Zig is a modern C alternative.

### Install

Zig itself:
```shell
brew install zig
```

zig-mode for emacs can be installed via: ```M-x install-package```


### Usage
See [main.zig](./main.zig) for an example. Run it with:

```shell
zig run main.zig
```

Run tests with:
```shell
zig test test_playground.zig
```
